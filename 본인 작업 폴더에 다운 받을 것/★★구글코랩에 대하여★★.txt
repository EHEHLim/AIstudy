< 구글 코랩(Google Colab)에 대하여 >

- 구글에서 무료로 제공하는 웹 방식의 머신러닝, 딥러닝 제작 실행 툴

- 웹에서 실행하므로 본인 컴에 파이썬, 아나콘다, 텐서플로 등과 같은 머신러닝/딥러닝을 실행하기 위한 툴들을 설치해 놓을 필요 없음.

- 구글 지메일(OOO@gmail.com) 계정을 보유하고 있어야 함.

- colab.research.google.com 주소로 접속 

- 구글 코랩 편집기 글꼴 변경하는 방법
   1. 크롬 주소창에 chrome://settings/fonts 입력 후 엔터를 치거나, 또는 우측 상단 점 3개 클릭 > 설정 > 모양 > 글꼴 맞춤설정
   2. "고정폭 글꼴"을 원하는 글꼴로 선택. Verdana 글꼴이 있다면 이걸로 설정 추천함.


< 주의 사항>

- 본인 컴에 고성능의 GPU(그래픽카드)가 없어도 구글 코랩의 GPU를 사용하여 훨씬 빠른 시간에 프로그램 실행 가능. 
   그러나 많이 사용하면 CPU 버전으로 자동 사용 제한됨. (대략 하루에 최대 10시간 정도 GPU 사용 가능)

- 일정 시간 동안 구글 코랩 실행하지 않으면 연결 자동 종료됨. 하지만 입력한 프로그램은 구글 드라이브로 자동 저장됨.

- 사용 시 "다른 이름으로 저장" 기능이 없으므로 "드라이브에 사본 저장" 기능으로 저장해야 함.
  만약 이름만 변경하고 저장하면 이전 이름의 파일이 저장되어 있는 채로 새 이름의 파일이 생성되는게 아니라,
  이전 이름의 파일이 새 이름으로 변경만 되는 것임. 주의할 것.
